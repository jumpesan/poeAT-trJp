# Home

Welcome to the poeAT-trJp Wiki.

poeAT-trJp is a Japanese fork of Awakened PoE Trade for Path of Exile.

The project started as a personal tool and has been extended with additional features and improvements focused on Japanese users.

## Main Features

### Core Features

The following core features are inherited from Awakened PoE Trade.

- Price Check
- Modifier Search
- Trade Market Integration
- Item Information Overlay

Credit for these core features belongs to the original Awakened PoE Trade project and its contributors.

### Localization

- Enhanced Japanese translations
- Translation generation from Path of Exile game files
- Improved modifier and item translation coverage

### Additional Features

- Inventory Snapshot
- Stash Snapshot
- Item Value Tracking
- Map Run Timer

## Platform

- Windows only
- Current Status: Beta / Work In Progress
