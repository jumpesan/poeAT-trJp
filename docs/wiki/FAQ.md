# FAQ

## Is poeAT-trJp an official Awakened PoE Trade project?

No.

poeAT-trJp is an unofficial Japanese fork of Awakened PoE Trade.

It is not affiliated with the original Awakened PoE Trade project or its maintainers.

## Which platform is supported?

Windows only for now.

Other platforms are not currently supported or tested.

## Is this a stable release?

No.

The current release is beta / work in progress.

Features, behavior, and data formats may change.

## Which file should I download?

Use the installer version if you want the simplest setup.

Use the portable / unpacked version if you do not want to install the application.

## Windows shows a warning. Is that normal?

Unsigned beta builds may trigger Windows SmartScreen or other security warnings.

Only run the application if you understand and accept the risk.

## Price Check does not appear. What should I check?

Check the following first:

- Path of Exile is running
- poeAT-trJp is running
- The hotkey is correct
- The item is under the mouse cursor
- The overlay is not hidden by settings

## The hotkey does not work. What should I check?

Check the hotkey settings in poeAT-trJp.

If another application uses the same hotkey, change the hotkey and try again.

## Where should I report issues?

Use GitHub Issues after the repository is public.

# Japanese

## FAQ

## poeAT-trJp は Awakened PoE Trade の公式プロジェクトですか？

いいえ。

poeAT-trJp は Awakened PoE Trade の非公式な日本語化・機能拡張フォークです。

元プロジェクトおよびそのメンテナーとは関係ありません。

## 対応環境は？

現在は Windows のみ対応です。

その他の環境では動作確認していません。

## 安定版ですか？

いいえ。

現在はベータ版・開発中の位置づけです。

機能、動作、データ形式は今後変更される可能性があります。

## どのファイルをダウンロードすればいいですか？

一番簡単に使いたい場合はインストーラー版を使用してください。

インストールせずに使いたい場合は、インストール不要版 / 展開版を使用してください。

## Windowsの警告が出ます。正常ですか？

署名されていないベータ版のため、Windows SmartScreen などの警告が出る場合があります。

内容を理解し、自己責任で実行してください。

## 価格チェックが表示されません。何を確認すればいいですか？

まず以下を確認してください。

- Path of Exile が起動している
- poeAT-trJp が起動している
- ショートカットキーが正しい
- アイテムにマウスカーソルが合っている
- オーバーレイが設定で非表示になっていない

## ショートカットキーが効きません。何を確認すればいいですか？

poeAT-trJp のショートカット設定を確認してください。

他のアプリとショートカットキーが重複している場合は、別のキーに変更して再度試してください。

## 不具合はどこに報告すればいいですか？

リポジトリを公開後、GitHub Issues から報告してください。
