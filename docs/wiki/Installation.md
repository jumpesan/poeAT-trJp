# Installation

This page explains how to install poeAT-trJp on Windows.

## Before You Start

poeAT-trJp is currently provided for Windows only.

This project is in beta. Features and behavior may change.

## Download

Download the latest release from the GitHub Releases page.

The release may provide two types of files:

- Installer version
- Portable / unpacked version

## Installer Version

Use the installer version if you want the simplest setup.

Basic flow:

1. Download the installer file from Releases.
2. Run the installer.
3. Follow the installer steps.
4. Start poeAT-trJp.

## Portable / Unpacked Version

Use the portable version if you do not want to install the application.

Basic flow:

1. Download the `win-unpacked` zip file from Releases.
2. Extract the zip file.
3. Open the extracted folder.
4. Run the application executable.

## Windows Security Warning

Windows may show a security warning such as SmartScreen for unsigned beta builds.

Only run the application if you understand and accept the risk.

## Next Step

After installation, continue to Quick Start.

# Japanese

## インストール

このページでは、Windows環境で poeAT-trJp を導入する方法を説明します。

## はじめる前に

poeAT-trJp は現在 Windows のみを対象にしています。

本プロジェクトはベータ版です。機能や動作は今後変更される可能性があります。

## ダウンロード

最新版は GitHub の Releases ページからダウンロードします。

Releaseには主に以下の2種類があります。

- インストーラー版
- インストール不要版 / 展開版

## インストーラー版

一番簡単に導入したい場合はインストーラー版を使用してください。

基本的な流れ:

1. Releases からインストーラーをダウンロードします。
2. インストーラーを実行します。
3. 画面の案内に従ってインストールします。
4. poeAT-trJp を起動します。

## インストール不要版

アプリをインストールせずに使いたい場合は、インストール不要版を使用してください。

基本的な流れ:

1. Releases から `win-unpacked` のzipファイルをダウンロードします。
2. zipファイルを展開します。
3. 展開したフォルダを開きます。
4. アプリの実行ファイルを起動します。

## Windowsの警告について

署名されていないベータ版のため、Windows SmartScreen などの警告が表示される場合があります。

内容を理解し、自己責任で実行してください。

## 次にやること

インストールが終わったら Quick Start を確認してください。
